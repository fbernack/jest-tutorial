## Install Jest

To install Jest as a dev dependency, execute:

```shell
npm install jest --save-dev
```

Add a corresponding run script to the `package.json`:

```json
{
  "scripts": {
    "test": "jest"
  }
}
```

Now you can run tests with:

```shell
npm test
```
